# Mommy Birthday Card

A cheerful, kid-friendly writing game made for iPad-style play.

The game helps a young child build a birthday card one step at a time with:

- big tap targets
- warm parent-style prompts
- emoji-supported word choices
- a send-card animation
- a party-building reward scene with fireworks

## Files

- `story-garden-game.html` - the complete game in one file

## How to Use

1. Open `story-garden-game.html` in a browser.
2. Tap word choices or type in the writing box.
3. Finish the card, send it, then build the birthday party.
